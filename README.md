# Simple Interest Calculator

This is a web-based tool that helps users calculate simple interest easily. 

### What it does:
* Takes the principal amount from the user.
* Allows users to select an interest rate using a slider.
* Calculates the total interest earned over a specific number of years.

### How to use:
1. Enter the amount you want to save.
2. Move the slider to pick your interest rate.
3. Choose the number of years.
4. Click "Compute Interest" to see your results!

### Files in this project:
* `index.html`: The structure of the calculator.
* `style.css`: The design and colors.
* `script.js`: The math logic.
* `simple-interest.sh`: A bash script version of the calculator.
